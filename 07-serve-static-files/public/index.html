Serving Up Even More

Most websites are way more than just 'index.html'. This page, for example, uses the link tag to reference an external style sheet styles.css, uses the script tag to reference an external JavaScript file, and uses several img tags to reference several jpg images. Each of these tags will result in an additional GET request sent to the server for the contents of the referenced file. With that in mind, this server is written to handle several different types of files.

The overall look of the site is controlled by the style sheet. The digital clock below is controlled by some client side JavaScript. The images, of course, are external files.
