hello-world
-----------

Outputs the simple greeting "Hello World from NodeJS!" to the command line.

Try It
------

This first sample shows that you can use Node to run code contained in a source file. This is done
by running the "node" command on the command line and specifying the source JS file as a command
line parameter.

  C:\NodeJS\01-hello-world>node hello-world
  Hello World from NodeJS!

  C:\NodeJS\01-hello-world>
